代码 客栈

乔布斯说:“我觉得每个人都应该学习一门编程语言,学习编程教你如何思考。”

代码 客栈 是一个基于Flask的个人博客网站，主要用于记录个人的学习笔记和心得体会。

本站使用的技术栈包括：

  • Python: Python 是一种易学易用的高级编程语言，广泛应用于Web开发、数据分析和人工智能领域。
  • JavaScript: JavaScript 是一种用于网页开发的脚本语言，可以为网页增加动态效果和交互功能。
  • HTML/CSS: HTML 是用于构建网页结构的标记语言，而 CSS 则用于设计和样式化网页。
  • SQLAlchemy: SQLAlchemy 是一个 Python 的对象关系映射（ORM）库，用于操作数据库。
  • Flask: Flask 是一个轻量级的 Python Web 框架，可以用于构建各种类型的 Web 应用程序。
  • Bootstrap: Bootstrap 是一个用于构建响应式网站的前端框架，可以用于设计和样式化网页。
  • Markdown: Markdown 是一种轻量级的标记语言，用于书写格式化文本。

本站的源代码可以在 GitHub 上找到。
